## 🔧 Easy Customization

Want to tweak the colors? Just edit the CSS variables in `static/css/style.css`:

```css
:root {
    /* Make it darker brown */
    --primary: #7d5f3d;
    
    /* Make it lighter cream */
    --bg-primary: #fffef9;
    
    /* Change amber to blue */
    --secondary: #3b82f6;
}
```
